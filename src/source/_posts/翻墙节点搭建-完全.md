## 安装 nginx

```bash
sudo apt install nginx -y
sudo ufw allow 'Nginx Full'
```

等待安装完成后

```bash
sudo vim /etc/nginx/nginx.conf
```

按a编辑

将user后面改为root

即:

```conf
user: root
```

在http内加入如下内容

```conf
server {
        listen 80;
        server_name 你的域名;
        root /root/www/pages;
        index index.html;
}
```

按Esc, 输入:wq回车以退出

随后

```bash
mkdir -p ~/www/pages/ && vim ~/www/pages/index.html
```

将以下内容完整的粘贴进去

按a编辑

## 安装Xray

```bash
wget https://github.com/XTLS/Xray-install/raw/main/install-release.sh
sudo bash install-release.sh
vim ~/cert/xray-cert-renew.sh
```

按a键

讲一下内容复制进去
``` bash
#!/bin/bash

/home/vpsadmin/.acme.sh/acme.sh --install-cert -d 你的域名 --ecc --fullchain-file /root/cert/xray.crt --key-file /root/cert/xray.key
echo "Xray Certificates Renewed"

chmod +r /root/cert/xray.key
echo "Read Permission Granted for Private Key"

sudo systemctl restart xray
echo "Xray Restarted"
```

按Esc, 输入:wq, 回车

继续
```bash
chmod +x ~/cert/xray-cert-renew.sh
crontab -e
```

按a键

加入以下内容

```
# 1:00am, 1st day each month, run `xray-cert-renew.sh`
0 1 1 * *   bash /root/cert/xray-cert-renew.sh
```

按Esc, 输入:wq, 回车

``` bash
xray uuid # 复制这里给出的uuid
sudo vim /usr/local/etc/xray/config.json
```

## 客户端配置

打开v2rayN

点击左上角服务器

选择 "添加[Vless]服务器"

- 别名随便填

- 地址填你的域名

- 端口填你自己设的

- 用户id填xray生成的

- 流控选xtls-rprx-version

- 传输层安全选tls

- 跳过证书验证选false

- 其余保持不变
